/* What is local storage? */

  • Type of web storage storing data as key-value pairs that can be accessed via JavaScript
  • Website and browser specific data that is stored locally on userâ€™s machine with no expiration date.
  • The data will persist even after the browser window has been closed.

/* Working with local storage */

As can be seen from the code sample, the interface for accessing and modifying contents of local storage is fairly simple. To learn more about local storage and its sibling technology session storage, visit MDN web docs on Web Storage API.

// Set a key-value pair
localStorage.setItem('users-name', nameValue)

// Access a value using key
localStorage.getItem('users-name')

// Delete a key-value pair
localStorage.removeItem('users-name')

// Remove all key-value pairs (associated with this domain)
localStorage.clear()
